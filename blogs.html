Var vs Let vs Const

Var : User can re-declare variable using var and user can update var variable. Var can declare without initialization.

Let :Users cannot re-declare the variable defined with the same let keyword but can update it., it does not initialize with undefined just like a var variable and return an error.

Const :Users cannot change the properties of the const object, but can change value of properties of the const object. Var cannot declare without initialization.

Arrow Function vs Normal Function

Arrow Function : Arrow functions do not have an arguments binding. However, they have access to the arguments object of the closest non-arrow parent function. Named and rest parameters are heavily relied upon to capture the arguments passed to arrow functions.

Regular Function : In a Regular function, there is an arguments local variable. If the number of arguments to our function is dynamic, using arguments allows us to easily do things like calculating the maximum number of passed arguments is an array-like object

Difference Between => Map, Foreach, Filter, Find

map : map run it against every element on the array but whats makes it unique is it generate a new array based on your existing array.

forEach : forEach just loop over the array and executes the callback

filter : filter executes the callback and check its return value.

find : find executes the callback and check it but return the first value completed condition.

Tamplet String

Tamplet String : Tags allow you to parse template literals with a function. The first argument of a tag function contains an array of string values. The remaining arguments are related to the expressions.
The tag function can then perform whatever operations on these arguments you wish, and return the manipulated string.
(Alternatively, it can return something completely different, as described in one of the following examples.)
The name of the function used for the tag can be whatever you want.
