Tamplet String

Tamplet String The advantage of backtick literals over quotation marks is when it is necessary to insert variables into a string phrase.
The other advantage of backtick literals (` `) is not needing to escape double quote or single quote characters using a backslash. if we create a multi-line string in JavaScriptThe solution would be to add \n newline characters, but that would make the string almost impossible to read.
Backtick literals solve this problem and are preferable for HTML. Template literals make clean JavaScript code when requesting an API
